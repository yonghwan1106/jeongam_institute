## 3. 게시글 작성하는 법

### 새 글 쓰기

1. 좌측 **게시글** 클릭
2. 우측 상단 **+ 새 글** 또는 `Create` 클릭
3. 다음 항목 입력:

| 항목 | 설명 | 필수 |
|---|---|---|
| **제목** | 글의 제목 (120자 이내) | ✅ |
| **URL 슬러그** | 자동 생성됨. 영문/숫자 권장 | ✅ |
| **분류** | 공지/언론/활동/에세이 중 선택 | ✅ |
| **요약** | 카드에 보일 1~2줄 요약 (200자 이내) | 권장 |
| **대표 이미지** | 카드 썸네일로 쓰임 (16:9 권장) | 권장 |
| **본문** | 글 내용 — 굵게/제목/인용/리스트/이미지 삽입 가능 | 권장 |
| **발행일** | 기본값은 오늘. 수동 조정 가능 | ✅ |
| **원문 링크** | 언론 보도일 때만 표시됨 | 보도시 |

### 발행하기

1. 우측 하단 **Publish** 클릭
2. 1분 이내 사이트에 자동 반영됨 (홈 "최근 활동" 섹션)

### 비공개 저장 (임시저장)

- 그냥 입력만 하고 Publish 누르지 않으면 **Draft**로 저장됨
- 추후 이어쓰기 가능

## 4. 답사 등록하는 법

1. 좌측 **답사** 클릭 → 새 항목 생성
2. 필수: **답사명**, **답사일**, **URL 슬러그**
3. 선택: **지역**, **한 줄 소개**, **대표 이미지**, **현장 사진**(여러 장), **답사 후기**, **참가 인원**
4. Publish

→ 사이트 `/activities/pilgrimage` 에 자동 노출 (날짜 최신순)

---

## 5. 강의 등록하는 법

1. 좌측 **강의·아카데미** 클릭
2. 필수: **강의명**, **트랙**(한국사/세계사/특강/온라인)
3. 권장: **회차** (예: 제1기 3강), **일정** (예: 2026.06.07 토 14:00), **강사**, **장소**
4. 추가: **포스터 이미지**, **유튜브 영상 링크**, **신청 링크**(구글폼 등)
5. Publish

---

## 6. 이미지 업로드 팁

- 드래그&드롭으로 업로드
- **alt 텍스트**(대체 텍스트) 꼭 입력 (시각장애인·검색엔진 모두 도움)
- 사진은 자동으로 여러 크기로 변환됨 — 원본 그대로 업로드해도 OK
- 권장 해상도: 1600×900 이상

## 7. 자주 묻는 질문

### Q. 글을 수정하면 사이트에도 바로 반영되나요?
A. 발행 후 **최대 1분** 이내 반영됩니다. 캐시 갱신 주기입니다.

### Q. 글을 잘못 발행했어요. 비공개로 돌리거나 삭제하려면?
A. 해당 글 열기 → 우측 상단 **⋯ 메뉴** → **Unpublish** (비공개) 또는 **Delete** (삭제)

### Q. 휴대폰으로도 글을 쓸 수 있나요?
A. 가능합니다. 모바일 브라우저에서 `/studio` 접속 → 로그인 → 동일하게 사용. 단 컴퓨터가 훨씬 편합니다.

### Q. 사진을 여러 장 한 번에 올리고 싶어요.
A. 답사 등록 시 **현장 사진** 필드에서 여러 장 동시 업로드 가능합니다.

### Q. 카페에 올린 글을 자동으로 가져올 수 있나요?
A. 자동 이전은 지원 안 합니다. 중요한 글만 골라 옮겨오시는 걸 권장합니다. (가벼운 글은 카페에 그대로, 자료성 글만 홈페이지로)

### Q. 백업이 되나요?
A. Sanity가 모든 글의 수정 이력을 자동 저장합니다. 글 우측 상단 **Review changes** 에서 과거 버전 확인·복원 가능.

| **후원/신청** | ✅ | — |

→ 사이트는 **공식 기록·외부 노출용**, 카페는 **회원 커뮤니티용**

### 글 작성 가이드

- **제목**은 검색이 잘 되도록 구체적으로 (예: "강의 안내" ❌ → "제2기 한국사 아카데미 강좌 모집 (5월 8일~6월 26일)" ✅)
- **요약**은 카드에서 잘리지 않도록 100자 내외 권장
- **이미지**는 가능한 한 첨부 (썸네일 있는 글이 클릭률 3배 높음)
- **분류** 정확히 — 검색·필터링에 사용됨

## 9. 문제 해결

| 증상 | 해결 |
|---|---|
| Studio 로그인이 안 됨 | Administrator(원장)에게 멤버 초대 요청 |
| 글이 사이트에 안 보임 | 1분 기다리기 → 그래도 안 보이면 발행 상태 확인 (Draft가 아닌 Published여야 함) |
| 이미지가 깨짐 | 파일 크기 5MB 이하인지 확인. 너무 크면 자동 압축 후 재업로드 |
| 한자가 안 나옴 | 본문에서 한자 직접 입력 가능 (예: 道學) — 자동 변환 안 함 |
